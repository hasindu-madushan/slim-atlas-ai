# slim-atlas (legacy TypeScript implementation)

This directory contains an older TypeScript implementation of `slim-atlas`,
built on Puppeteer and the Lightpanda browser. It is kept here for historical
reference only.

The active implementation is the Rust rewrite at the repository root, which
shares the same name and protocol. This directory is not part of the build and
is not published.

See `../DESIGN.md` for the architecture of the Rust implementation and
`../PLAN_PHASE_1.md` for the current implementation plan.
